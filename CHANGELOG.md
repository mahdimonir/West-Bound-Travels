Premium houseboat booking platform for Tanguar Haor and Sunamganj wetlands.

## Features

- Browse & book 4 luxury houseboats (2 AC, 2 Non-AC)
- Real-time availability & secure room selection
- SSLCommerz payments (bKash, Nagad, cards, etc.)
- Email-verified registration & OTP flow
- User dashboard (bookings, profile, invoices)
- Admin panel (fleet, destinations, blogs, static pages)
- Automated booking & status emails
- Responsive, luxury-themed UI
